# 이사갈 동네 리포트 구현 예외사항

## 제외 기능 (구현하지 않음)

### 1. 부동산 거래 및 투자 판단

- 특정 아파트, 빌라, 오피스텔 매수 추천
- 부동산 가격 상승/하락 예측
- 투자 수익률 계산
- 전세 사기 여부 단정
- 권리분석, 등기부등본 분석, 법률 자문
- 대출 가능 금액, 세금, 중개수수료 정밀 계산

### 2. 매물 및 계약 관련

- 부동산 매물 등록/조회
- 중개사 연결 또는 상담 예약
- 전자계약 기능
- 계약서 작성 및 검토
- 보증보험 가입 연동
- 임대인/임차인 신원 검증

### 3. 개인정보 및 사용자 계정

- 주민등록번호, 소득, 신용점수 등 민감정보 수집
- 개인 금융정보 연동
- 가족 구성원 상세 정보 저장
- OAuth, SNS 로그인
- 다단계 인증(2FA, OTP)
- 사용자별 장기 이력 관리

### 4. 지도 및 경로 고급 기능

- 상용 지도 API 기반 상세 지도 렌더링
- 실시간 길찾기
- 출퇴근 경로 최적화
- 실시간 교통 혼잡도 반영
- 3D 지도, 로드뷰, 항공사진
- 지적도 또는 토지 경계 정밀 표시

### 5. 고위험 지역 판단

- 범죄율 기반 지역 순위화
- 특정 지역에 대한 위험/안전 단정
- 학교 서열화 또는 학군 순위 산정
- 소음, 악취, 민원 등 공식 API로 확인하기 어려운 지표 단정
- 재난 위험도를 보험/금융 판단 수준으로 제공

### 6. 알림 시스템

- 푸시 알림
- SMS 알림
- 이메일 발송
- 카카오톡 등 메신저 알림
- 가격 변동 또는 신규 거래 알림
- 장기 모니터링 알림

### 7. 고급 분석 및 자동화

- 머신러닝 기반 가격 예측 모델
- 자동 이사 지역 추천 엔진
- 대규모 배치 데이터 웨어하우스 구축
- 민간 사이트 크롤링
- 유료 데이터 구매 및 연동
- 통계 모델 기반 인구 이동 예측

### 8. 운영 및 관리 고급 기능

- 다중 운영자 권한 관리
- 조직/테넌트 분리
- 과금 및 구독 관리
- 운영자 감사 로그
- SLA 모니터링 대시보드
- 데이터 품질 자동 검수 워크플로우

### 9. 리포트 출력 및 공유

- PDF 다운로드
- 엑셀 다운로드
- 외부 공유 링크
- 소셜 미디어 공유
- 리포트 워터마크 또는 브랜딩 편집
- 인쇄 최적화 화면
